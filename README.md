

## Build
### option 1 temporary install
~~~ shell
git clone https://github.com/Overionised/probability-tests
cd probability-tests
g++ main.cpp -o probability-tests
~~~

### option 2 permanent install
~~~ shell
git clone https://github.com/Overionised/probability-tests
cd probability-tests
g++ main.cpp -o probability-tests
sudo cp probability-tests /usr/bin/
~~~
## Run
### option 1 usage
~~~ shell
./probability-tests
~~~

### option 2 usage
~~~ shell
probability-tests
~~~
